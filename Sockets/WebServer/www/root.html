You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /bigLetters?str=<input>&font=<0 or 1> -- returns str in big word art, non-letters are interpreted as spaces. there are 2 available fonts. ex: /bigLetters?str=HELLO&font=0
  • /gFib?n=<int>&f0=<int>&f1=<int> -- returns fib(n) but with the chosen base cases for fib(0) and fib(1). ex: /gFib?n=10&f0=-3&f1=5
